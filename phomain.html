베트남쌀국수 전문점 포메인의 사이트입니다
쌀국수의 대중화에 앞장서 온 브랜드로 브랜드의 네임밸류가 있는만큼 브랜드 아이덴티티의 감도를 맞춰 포메인의 연혁과 메뉴소개, 사회공헌 내용을 담고 있습니다.

CLONE CODING

  • PURPOSE

    국내 최대 베트남쌀국수 체인점 포메인의 홈페이지로 회사의 사업소개, 회사의 연혁등을 보여주는 사이트입니다.

  • CONCEPT

    고급스럽고 퀄리티 높은 사진사용으로 고급 브랜드로 도약

  • POINT

    전체 풀페이지로 제작되었으며 스크롤에 따라 다음페이지로 이동하는 효과를 주었습니다.

  • OVERVIEW

    포메인의 웹사이트를 클론코딩 하였습니다. GD WEB 2019년 4월 WEB부문 선정작이며 안정적인 컨셉으로 사진과 초록색을 활용하여 표현하였습니다.

  • 사용 프로그램

    Visual Studio Code

  • CONTENT

    • - 총 누적판매량
    • - 전체 메뉴 소개
    • - 딜리버리 매장 / 카카오톡 선물하기
    • - 사회공헌
    • - SNS
    • - 가맹점 문의

  • 주요 학습 내용

    • - 전체 fullpage 적용
    • - 네비게이션 설정
    • - Visual 영역 슬라이드 적용
    • - 숫자 카운트 액션 적용
    • - 이미지와 폰트의 animation 적용
    • - img hover 할때 after 이미지 보이게 적용

PC MODE
