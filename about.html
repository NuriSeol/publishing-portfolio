Software Skill

이론수업과 클론코딩을 통해 html, css, javascript, jQuery 실무에 가까운 procss로 교육 받고 숙련시켰습니다. 디자인 편집 툴인 photoshop, illustrator, adobe XD, Figma를 활용하여 코드로 구현 전 확인할 수 있는 모든 퍼포먼스 구현 가능합니다.

  • HTML

    다양한 사이트의 클론 코딩을 통해서 box 구조의 HTML 원리를 이해하고 header, section, footer 등 다양한 태그를 기반으로 HTML5 구조 작성 가능합니다.

  • CSS

    font, color, background-color를 지정하고, position을 통한 위치를 지정할 수 있습니다 transition, animation을 바탕으로 동적인 효과 구현 가능합니다

  • JavaScript

    click, onmouseover 등의 event 메서드를 통해 사용자가 웹페이지에서 마우스 사용 시 화면 색상 변경, 이미지 전환 등의 동적인 효과 구현가능합니다.

  • JQuery

    addClass, removeClass 메서드를 통해 header 영역 높이값을 변경할 수 있으며 다양한 플러그인을 사용하여 슬라이드, 스크롤트리거를 적용할 수 있습니다.

  • Photoshop

    인물 보정, 상세 페이지 작업 등 편집다지인의 경력으로 포토샵을 익숙하게 다룰 수 있습니다.

  • Illustrator

    기본적인 툴은 모두 사용이 가능하며, 명함디자인, 배송안내문 제작 경험과 일러스트의 가장 중요한 펜툴을 사용하여 아이콘 제작 가능합니다.

  • XD

    포토샵과 같은 프로세스로 NCS 수업을 통해서 빠르게 습득했으며, HTML로 구현 전 페이지를 확인 할 수 있는 모든 퍼포먼스 구현 가능합니다.

  • InDesign

    단락스타일, 문자스타일 설정을 지정할 수 있고 자간 조정, 행간 조정 등 간격에 대해 이해하고 있습니다. pdf용 인터렉티브 기능을 넣을 수 있습니다.
