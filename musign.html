PC MODE

Image, Text, SVG 스크롤 애니메이션을 넣어 역동적인 느낌을 구현하였으며, 스크롤 위치에 따른 배경색 변경 효과를 추가하였습니다.

MOBIL MODE

Tablet Mode는 각 요소가 1줄에 1개씩 차지하도록 변경하였으며, 우측 중앙에 있던 메뉴 버튼을 우측 상단으로 변경하였습니다.

TABLET MODE

Mobile Mode는 Tablet Mode의 전체적인 틀을 유지하되, 가로로 늘어져있던
